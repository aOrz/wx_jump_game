# 微信跳一跳自动玩耍工具

## 安装

先安装 node 和 一些依赖，

安装 `adb`

canvas 依赖

OS X `brew install pkg-config cairo libpng jpeg giflib`

Windows `https://github.com/Automattic/node-canvas/wiki/Installation---Windows`

## 开始

手机连接到电脑上，打开调试，使用 adb 命令可以连接上时，执行本（node wx_bump.js）脚本即可。
